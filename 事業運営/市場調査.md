# Casinohub — アミューズメントカジノ市場調査

> 調査日: 2026-03-02 / 情報源: PR TIMES, ビジネスジャーナル, カジノジャパン, 業界知見

---

## 1. 市場規模

### 店舗数
- 2025年1月時点: アミューズメントポーカールーム全国 **411店舗**（Windstorm Media調査）
- 別ソース: 全国 **600店舗超え**（ビジネスジャーナル報道）
- 2018年頃は全国100店舗程度 → 5〜6年で4〜6倍に急増
- 東京だけで推定100〜150店舗（新宿・渋谷・池袋・秋葉原・六本木に集中）

### 市場規模（金額）
- 公式統計なし。推定 **数百億円規模**（店舗売上ベース）
- 1店舗あたり月商: 小規模50〜150万円 / 中規模200〜500万円 / 大規模1,000万円+
- 参考: パチンコ市場が約14〜15兆円。極めてニッチだが成長率は高い

### プレイヤー人口
- 2021年: 約88万人
- 2025年: 推定 **100万〜200万人**（オンライン含む）
- アミューズメントカジノに定期的に通うアクティブプレイヤー: 推定10〜30万人

### 成長要因
- YouTube/TikTokでのポーカー動画の影響（ヨコサワ等）
- オンラインポーカーアプリ（GGPoker, KKPoker）の日本語対応
- WSOPでの日本人プレイヤーの活躍
- コロナ禍後の「リアル体験需要」回復
- 大阪IR（統合型リゾート）開業への期待

## 2. 業態分類

### (A) ポーカールーム（専門型）— 最多
- テキサスホールデム特化
- リングゲーム + トーナメント + SNG
- 客単価: 2,000〜5,000円/回
- 収益: エントリーフィー、時間制チップ購入、ドリンク
- 例: ROOTS、Spadie系列、Hand's Poker Room

### (B) カジノバー（飲食併設型）
- バー営業 + 複数カジノゲーム（BJ, ルーレット, バカラ, ポーカー）
- 客単価: 3,000〜8,000円（飲食込み）
- 飲食店営業許可 + 深夜酒類提供飲食店営業届出の組み合わせ
- 個人経営が多い

### (C) 総合アミューズメントカジノ（大型施設型）
- カジノゲーム全般を大規模に提供
- 客単価: 3,000〜10,000円
- VIP会員制度、イベント、ディーラースクール併設も
- 例: GUTSHOT、AKIBAギルド、カジノヴィーナス

### (D) ハイブリッド型（オンライン連動）
- GGPoker/WPTなどの海外トーナメント予選を店舗で開催
- 増加傾向

## 4. 法規制

### 風営法 第4号営業（遊技場営業）

| 要件 | 詳細 |
|---|---|
| 人的要件 | 成年、欠格事由非該当（破産者・前科等NG） |
| 構造的要件 | 客室9.5m²以上、照度10ルクス以上 |
| 場所的要件 | 学校・病院等から50〜100m以上離れていること（都道府県条例） |
| 営業時間 | **原則0時まで**（地域により1時） |
| 管理者 | 選任必須 |

### 絶対的な制約
- **換金・景品交換は一切不可**（賭博罪に抵触）
- **トーナメント賞品**: 上限は「おおむね800〜1,000円程度」（解釈に幅あり。海外渡航パッケージの扱いはグレー）
- **深夜営業不可**: 業界最大のボトルネック。金曜深夜の需要が取れない

## 5. 競合ツール・システム — **ほぼ皆無**

| カテゴリ | 現状 |
|---|---|
| チップ管理 | **Excel / 紙** が主流。専用ツールなし |
| POS | Airレジ・スマレジ・Square等の汎用POSを転用 |
| トーナメント管理 | Poker Tournament Manager（海外製PC用） |
| 会員管理 | LINE公式 + Google Sheets |
| 予約管理 | TableCheck / RESERVA等の汎用ツールを転用 |
| 統合管理 | **存在しない** |

**→ これがCasinohubの最大のチャンス。チップ管理・POS・トーナメント・会員管理を一元化するSaaSは市場に存在しない。**

## 6. 業界の課題（現場の声）

### 法規制
- 「深夜0時で閉めないといけないのが一番痛い。金曜の夜、テーブルが全部埋まっているのに閉店」
- 賞品規制の曖昧さ（地域・警察によって解釈が異なる）
- 無許可営業の横行が業界イメージを悪化させている

### 運営・管理
- 「チップの棚卸しが毎日地獄。数が合わないとスタッフ全員で何時間も数え直す」
- 「トーナメントの集計をExcelでやっているが、ミスが多くてクレームになる」
- 「会計がバラバラ。チップ購入・ドリンク・トーナメント参加費が別々の管理で、月末の集計がカオス」
- 「LINE公式で予約を受けているが、当日キャンセルが多すぎて席が無駄になる」

### 人材
- 「ディーラーが辞めると代わりがいない。育成に最低3〜6ヶ月」
- 「バイト感覚で来る人は続かない」

### 競争
- 「隣に新しい店ができるたびに常連が流れる。差別化が難しい」
- 初心者が上級者との実力差で離脱するリテンション問題
- 女性客を取り込めない（雰囲気が男性向け）

## 8. Casinohubの市場機会まとめ

### ポジティブ
- 店舗数400〜600で急成長中、管理ツールは**完全な空白地帯**
- チップ管理の手作業が最大の現場ペイン → ここを解決すれば刺さる
- ポーカーブームが追い風、大阪IRでさらに関心↑
- 月額1〜3万円 × 300〜600店舗 = 年間数億円規模のTAM

### リスク
- ニッチ市場（パチンコ市場の1/1000以下）
- 法的グレーゾーンが多い業界
- 店舗の入れ替わりが激しい（チャーンリスク）
- 深夜営業規制により店舗の収益力自体が限定的
